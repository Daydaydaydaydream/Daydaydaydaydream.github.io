c++基本语法及STL

STL 被称作”标准模板库“，其中包含大量高效的算法以及常用的容器。我们可以通过 std:: 这个关键字来使用它，例如 std::vector 等。

类型

int 是整数类型，顾名思义只能用作整数变量，可以存储符号。

char 是单个字符。

float 是单精度，double 是双精度。float 的存储范围 ±2^128 小于 double ±2^1024，但是占据内存小。

bool 是布尔变量，常用与条件判断，可以动态转换 int 的0和1。

string 为字符串。

c++为静态语言，在程序运行前每个变量都需要赋予他们类型才能编译成功，这和 python 这种动态语言直接对变量赋值的方法有所不同。

那么在 python 中一个函数可以接受不同类型的参数，那在 c++ 中想要使用“一种函数”去处理不同参数我们可以使用重载（overload）去实现（其实我不太明白重载的意义，和创建两个函数似乎没有区别只是命名一样。目的是更加方便使用函数？）。

若继续学习可以了解到“模板”也能更高效实现这个需求。

c++11 标准及以后引入了 auto 关键字，它会更具你对这个变量初始化的值来设置变量类型（有点类似python 动态语言的使用方式），十分方便可以使得代码干净整洁。但是注意不要滥用，否则会导致可读性降低。

结构

什么是结构（struct）？结构是一组命名变量，每个变量都有他们的类型且允许程序员去捆绑他们在一起。

结构体可以传递组内变量的信息，同时也可以改变组内变量的值。

std::pair 是一种关联容器，具有两个任意类型字段的 STL 内置结构体。
